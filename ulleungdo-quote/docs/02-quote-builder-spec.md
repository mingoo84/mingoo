# 견적 작성 탭 — 기능 명세서

> 대상: `ulleungdo-quote/index.html`의 **견적 작성(quote)** 탭
> 기준일: 2026-07-06 · 코드 기본값 기준 (요금은 화면에서 수정 후 "요금 저장" 시 해당 브라우저에 계속 유지됨)

---

## 1. 개요

일자·인원·선박·투어·숙박·식대를 입력하면 **1인당/등급별 금액과 총액**이 실시간 계산되고,
**견적서 문구(카톡/문자용 텍스트)** 와 **견적서 이미지(PNG)** 가 자동 생성되는 화면.
생성한 견적은 이력(최근 8건)과 예약 관리로 넘길 수 있다.

```
[좌측 입력]                         [우측 결과]
① 고객 정보                         견적 요약 티켓 (노선·일정·인원·시즌·총액·계약금·내역)
② 운항 시간표 · 선박 · 승선권          최근 견적 이력 (8건, 탭하여 복원)
   ③ 일자 · 시즌
   ④ 독도 탐방
③ 투어방법                          [하단 출력]
④ 숙소                              완성된 견적서 문구 (복사/문자 발송)
⑤ 식대                              견적서 이미지 미리보기 (PNG 저장)
⑥ 인원 구성
```

---

## 2. 입력 필드 상세

### ① 고객 정보
| 필드 | 형식 | 규칙 |
|---|---|---|
| 고객명 | 텍스트 | **필수** (견적 생성 시 검증) |
| 담당자 | 선택 | 서민구 팀장 / 이재훈 팀장 |
| 담당자 연락처 | 자동 | 담당자 선택에 따라 자동 (둘 다 010-3361-4142) |
| 인원(총 팀 인원) | 숫자 | 기본 = 인원 구성 합계 자동. 직접 수정 시 "N인팀" 문구에만 반영(금액은 인원 구성 기준) |

### ② 운항 시간표 · 선박 · 승선권
| 필드 | 형식 | 규칙 |
|---|---|---|
| 선사 시간표 링크 3종 | 링크 카드 | 울릉크루즈/대저해운/씨스포빌. 출발일 입력 시 해당 날짜·월로 딥링크 자동 세팅 |
| 가는편(출항지) | 선택 | 포항크루즈(crz) / 포항쾌속(poh) / 묵호(muk) |
| 오는편(입항지) | 선택 | 기본 "가는편과 동일"(자동 동기화). 직접 다른 선박 선택 가능. 같은 선박 재선택 시 자동 동기화 복귀 |
| 왕복 승선권(대인 1인) | 금액 | 선박 선택 시 기본값 자동 입력, 수정 가능 |
| 유류할증료(1인) | 금액 | 할인 없이 전 인원 1인당 가산, 수정 가능 |

### ③ 일자 · 시즌
| 필드 | 규칙 |
|---|---|
| 일정 | 1박 2일 / 2박 3일 |
| 출발일 | **필수**. date 입력 |
| 도착일 | 자동 = 출발일 + 박수 (크루즈 규칙 시 +1일 추가) |
| 요금 시즌 | 주중/주말/성수기 필. 출발일 기준 자동 추천(금·토=주말, 7/15~8/20=성수기), 수동 변경 가능 |

### ④ 독도 탐방
| 필드 | 규칙 |
|---|---|
| 탐방 선박 | **미선택**(기본) / 썬라이즈호 / 씨스타1호 / 독도드림호 — 선박을 고르면 독도 코스 포함, 미선택이면 아래 필드 전부 비활성(회색) |
| 시간대 | 오전/오후 (출항 시간 입력 시 자동 판별) |
| 출항 시간 | 시(1~12)/분(5분 단위) 드롭다운 |
| 탐방 인원 | 기본 = 전체 인원(전원 참여). 줄이면 그 인원만 과금 |
| 독도 1인 추가요금 | 금액, 수정 가능 |

### ③′ 투어방법
| 필드 | 규칙 |
|---|---|
| 투어 방법 | 버스투어 / 택시투어 / 스타렉스투어 / 단독콤비투어 / 단독25인승 / 단독35인승 / 자유렌트 |
| 투어 요금 | 버스투어 = **1인당** 라벨, 나머지 = **팀 총액** 라벨. 수정 가능 |
| (자유렌트 시) 차종 | 레이/아반떼·K3/코나·셀토스/투싼·스포티지/스타리아·스타렉스/카니발/쏠라티 |
| (자유렌트 시) 시즌 | 비수기/성수기 — 요금 시즌에서 자동(성수기만 peak), 수동 선택 가능 |
| (자유렌트 시) 차량대수 | 기본 1 |
| (자유렌트 시) 차량기준가 | 차종·시즌 기본값 자동, 수정 가능 |

### ④′ 숙소 (여러 행 추가 가능)
| 열 | 규칙 |
|---|---|
| 유형 | 모텔/호텔/리조트/펜션 |
| 숙소 이름 | 자동완성: 유형별 실숙소 목록 (커스텀 드롭다운) |
| 방종류 | 자동완성: 숙소별 실객실 목록 |
| 객실단가 | 알려진 숙소·객실은 현재 시즌 단가 자동 채움, 수정 가능 |
| 객실수 | 기본 1 |

### ⑤ 식대 (1인 기준)
| 필드 | 규칙 |
|---|---|
| 조식/중식 수량 | **자동**(읽기전용) — 아래 끼니 규칙 참조 |
| 크루즈 조식 | 가는편이 포항크루즈일 때만 표시: 크루즈조식/일반조식 선택 (조식 1끼에 적용) |
| 석식 수량 | 자유식 — 기본 0, 직접 입력 |
| 끼니별 1인 단가 | 조식/크루즈조식/중식/석식 각각 수정 가능 |

**끼니 자동 규칙**
| 일정 | 선박 | 조식 | 중식 | 합계 |
|---|---|---|---|---|
| 1박2일 | 전 선박 | 1 | 2 | 3식 |
| 2박3일 | 크루즈·포항쾌속 | 2 | 3 | 5식 |
| 2박3일 | **묵호** | 2 | 2 | **4식** |

### ⑥ 인원 구성 (7등급)
| 등급 | 기본 할인군 | 승선권 할인율 |
|---|---|---|
| 대인 | 할인 없음 | 0% |
| 경로 | 20% 할인 | 20% |
| 소인 | 50% 할인 | 50% |
| 단체 | 10% 할인 | 10% |
| 장애 | 50% 할인 | 50% |
| 장애보호 | 20% 할인 | 20% |
| 보훈대상자 | 20% 할인 | 20% |

- 각 행에서 할인군(없음/10/20/50%) 드롭다운으로 변경 가능
- 각 행에 **1인당 금액 실시간 표시** (승선권 할인 적용 + 유류할증)
- 할인은 **승선권에만** 적용 — 유류할증·투어·식대·독도·숙소는 할인 없음

## 3. 금액 계산 규칙

```
등급별 1인당 = round100( 승선권 × (1 − 할인율) ) + 유류할증료
등급 소계   = 등급별 1인당 × 등급 인원

투어비 (별도 라인)
  · 버스투어(인당제)  = 1인당 요금 × 실효인원        ※ 실효인원 = max(인원구성 합계, 직접 수정한 총 팀 인원)
  · 그 외(총액제)     = 팀 총액 그대로 (인원 무관)    ※ 1인당 환산액은 참고 표기
  · 자유렌트          = (차종 시즌 기준가 + 마진 10,000) × 박수 × 차량대수

독도 탐방 (별도 라인) = 독도 1인 추가요금 × 탐방 인원
숙소 (별도 라인)      = Σ (객실단가 × 객실수)
식대 (별도 라인)      = [조식비 + 중식단가×중식수 + 석식단가×석식수] × 전체 인원
  · 크루즈편 조식 1끼는 선택(크루즈조식/일반조식) 단가, 나머지 조식은 일반조식 단가

총액   = Σ등급 소계 + 투어비 + 독도 + 숙소 + 식대
계약금 = 인원 구성 합계 × 100,000원
부가세 = 카드/현금영수증 시 10% 별도 (문구 자동 포함)
round100 = 100원 단위 반올림
```

**크루즈 전야 출항 규칙**: 가는편이 포항크루즈면 전날 밤 출항(선상 1박)이므로
표기 박수 +1 (예: 2박3일 → 3박4일), 도착일 +1일, 일정표 맨 앞에 "전일 밤 승선" 행 추가. **요금은 원래 박수 기준 그대로.**

## 5. 견적서 출력

### 5-1. 문구 (텍스트)
```
현지 여행사 굿투어  {선박표기}   울릉도여행 일정 및 견적서
수신 {고객명}님   담당 {담당자}   연락처 {연락처}
발신 굿투어

1. 건   명 : {고객명}님 {N}인팀 {선박표기} {투어명}{+독도}
2. 기   간 : {출발일(요일)} ~ {도착일(요일)} ({N박N+1일} · {시즌}{· 크루즈 전야출항})
3. 제   공 : 굿투어주식회사(사업자: 506-81-87523) / 계좌: 농협 351-0693-3158-03
4. 포   함 : 왕복승선권 + {숙소 나열} + {투어명}{+ 독도 탐방(선박·시간·인원)}
5. 청구금 : 총금액 {총액} {한글금액}정
   - {등급} {n}명 × 1인 {단가} = {소계}      ← 등급·투어·독도·숙소·식대 라인별
   * {총액제 투어 1인당 환산 참고}
   * 유류할증료 1인 {금액} 포함              ← 0원이면 생략
   * 카드/현금영수증 시 부가세 10% 별도. 계약금 1인당 100,000원, 총 {계약금}

{일정표: 일자/시간/내용/비고 — 탭 구분}

 ◆ 상기 일정은 현지 상황 및 기상에 따라 변경될 수 있습니다.
 ◆ 정확한 승선 가능일은 선사 홈페이지에서 최종 확인 후 예약을 확정해드립니다.
```
- **복사** 버튼(클립보드) · **문자로 보내기** 버튼(sms: 링크, iOS/Android 대응)

### 5-2. 이미지 (PNG)
회사 양식 문서(820px): 네이비 헤더(현지 여행사 굿투어 · 울릉도여행 일정 및 견적서) → 수신/담당/연락처 표 → 건명~청구금 요약 → 일정표(일자 rowspan 병합) → 회사·계좌 푸터. html2canvas 2배율로 `굿투어_견적서_{고객명}_{출발일}.png` 저장.

## 6. 검증 · 저장 · 연동

| 항목 | 규칙 |
|---|---|
| 생성 차단 검증 | 고객명 없음 / 출발일 없음 / 인원 합계 0 → 토스트 + 해당 입력으로 스크롤 |
| 요금 저장 | "요금 저장" 버튼(선박·투어·독도 섹션 3곳, 동일 동작) → 승선권·유류·투어·차종·독도·식대 단가 일괄 저장. 수정 후 미저장 시 "미저장" 배지 |
| 자동 저장(draft) | 모든 입력 변경 시 자동 저장, 새로고침 시 복원. 단 **요금값은 draft가 아닌 요금 저장소에서만** 로드(이력이 요금을 덮어쓰지 않음) |
| 견적 이력 | 생성 시 최근 8건 저장(입력 스냅샷 포함). 탭 → 확인 후 전체 복원 |
| 예약 연동 | 예약관리 탭 "현재 견적을 예약으로 저장" → 고객명·일정·선박·인원·금액·스냅샷 저장(상태 대기/결제 미수 기본) |
| 사전정보 연동 | 고객 사전정보 링크(#load=…) 열람 시 고객명·출발일·일정·항구·인원 자동 반영 |
| 초기화 | 입력 초기화(확인창) — 요금 저장값·이력은 유지 |
