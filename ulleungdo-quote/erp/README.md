# 굿투어 ERP (1차 MVP)

울릉도 여행사 굿투어의 내부 운영 시스템. 고객·상담·견적·예약·입금·문자·매출분석을 한 화면 체계에서 처리한다.

- **접속**: 배포 주소 뒤에 `/erp/` (예: `https://<vercel도메인>/erp/`)
- **구조**: `index.html` 단일 파일 (프레임워크 없음), 데이터는 브라우저 localStorage
- **데모 데이터**: 첫 실행 시 자동 생성. 초기화는 로그인 화면 하단 또는 설정 → 데이터 관리

## 화면 구성

| 메뉴 | 내용 |
|---|---|
| 대시보드 | 신규문의·견적대기·입금대기·출발임박, 오늘 할 일, 미수금, 예약 캘린더 |
| 고객 | 목록·검색·중복알림, 상세 7탭(기본/상담/견적/예약/입금/문자/변경이력), 입력링크 발송 |
| 상담 | 오늘 재통화·견적필요 필터, 다음 연락 관리 |
| 견적 | 항목 빌더 + 우측 요약(판매가·원가·이익·마진율 자동계산), 미리보기(원가 숨김)·PDF·문자·예약 전환 |
| 예약 | 상태 필터·검색, 상세 9탭(여행자/배편/숙박/투어/입금/문자/이력), 배편 결항 → 상태·문자 연계 |
| 입금/정산 | 예약금·잔금·추가금·환불, 미수금 자동계산, 입금 후 상태변경 추천 |
| 문자 | 운영 템플릿 12종, 변수 자동 치환, 발송 이력 (광고성 문자 없음) |
| 매출분석 | 기간별 매출·입금·미수·순이익, 월별 추이, 예약별 손익표, CSV 다운로드 |
| 설정 | 직원·권한, 회사정보·문구, 상품, 데이터 백업/복원 |

## 고객 모바일 입력폼

고객 상세 → 입력링크 생성 → 문자 발송. 고객이 `#/form/<토큰>` 링크에서 일정·인원·희망사항을 입력하면 상담(견적필요)으로 자동 접수된다. 링크 유효기간·개인정보 동의 검증 포함.

## 데이터 주의

- 데이터는 **이 브라우저에만** 저장된다. 직원 간 실시간 공유는 아직 안 됨.
- **설정 → 데이터 관리 → JSON 백업**을 주기적으로 내려받아 보관할 것.
- 서버 저장·실제 문자 발송 계획은 [BACKEND-DESIGN.md](./BACKEND-DESIGN.md) 참고.
