# 🏝️ 굿투어 울릉도 견적 계산기

> 클릭 몇 번으로 끝나는 울릉도 여행 견적서 — 여행사 직원용 원파일 웹앱

일자·인원·선박·투어·숙소만 고르면 **금액 계산 → 견적서 문구 → 견적서 이미지(PNG)** 까지 자동으로 완성됩니다.
완성된 문구는 카카오톡/문자에 바로 붙여넣고, 이미지는 PNG로 저장해 전송하세요.

**설치 없음. 서버 없음. `index.html` 파일 하나를 브라우저로 열면 끝.**

---

## ✨ 주요 기능

### 💰 견적 계산
- **요금 등급 7종** — 대인 / 경로 / 소인 / 단체 / 장애 / 장애보호 / 보훈대상자
  - 등급별 **1인당 금액 실시간 표시**
  - 등급마다 할인군(할인 없음 · 10% · 20% · 50%)을 드롭다운으로 즉시 변경 가능
- **등급별 소계 + 총액 구분 표기** — `대인 2명 × 1인 361,000원 = 722,000원` 형식
- 한글 금액 병기 (`삼십육만천원정`) · 계약금 자동 산출 (1인당 100,000원)

### 🚢 선박 · 일정
- **가는편/오는편 선사 분리 선택** — 출항지와 입항지가 달라도 OK
  - 예: 포항크루즈로 들어가서 묵호(씨스포빌)로 나오기
  - 혼합 노선은 두 선사 왕복 요금의 평균(100원 단위 반올림)으로 계산
- 선사 3사 지원: 포항크루즈(울릉크루즈) · 포항쾌속(대저해운) · 묵호(씨스포빌)
- **선사 운항시간표 바로가기** — 출발일 입력 시 해당 날짜/월로 딥링크 자동 세팅
- 도착일 자동 계산 · 출발일 D-day 배지 · 날짜에 요일 표시
- **요금 시즌 자동 추천** — 금·토는 주말, 7/15~8/20은 성수기로 자동 제안 (수동 변경 가능)

### 🚌 투어 · 숙소
- 투어 4종: **버스투어 / 택시투어 / 콤비투어(버스+택시 평균가) / 렌트카투어**
- 렌트카투어 선택 시 **스노클링 추가 옵션** (+20,000원/인)
- **독도 코스 포함** 체크 한 번으로 독도 요금 자동 반영
- 숙소: **모텔/호텔/리조트/펜션** 유형 구분, **여러 개 추가 가능**, 각각 이름·추가요금 입력
  - 추가요금은 총액에 자동 합산 (0원이면 견적서에 표기만)

### 📋 출력
- **견적서 문구 자동 생성** — 카카오톡/문자에 바로 붙여넣는 완성형 텍스트 (일정표 포함)
- **견적서 이미지 PNG 저장** — 회사 양식(수신/담당/일정표/계좌) 그대로 이미지화
- 원클릭 복사 (최신 Clipboard API + 구형 브라우저 폴백)

### 💾 편의 기능
- **자동 저장** — 입력 중 브라우저가 꺼져도 이어서 작업
- **최근 견적 이력 8건** — 탭 한 번으로 과거 견적 통째로 불러오기 (덮어쓰기 전 확인창)
- 모바일 퍼스트 — 하단 고정 총액바, 44px 터치 타깃, 좁은 화면 1열 전환
- 입력 검증 — 고객명/출발일/인원 누락 시 생성 차단 + 해당 입력으로 자동 스크롤

## 🚀 사용 방법

1. 이 저장소를 클론하거나 `index.html` 파일만 다운로드
2. 브라우저(크롬 권장)로 열기 — 모바일/PC 모두 지원
3. 위에서 아래로 순서대로 입력 → **견적서 문구 생성** → 복사 or PNG 저장

> PNG 저장 기능은 html2canvas CDN을 사용하므로 최초 로딩 시 인터넷 연결이 필요합니다.

## 🔧 설정 변경 포인트

모든 설정은 `index.html`의 `<script>` 상단에 모여 있습니다.

| 상수 | 용도 |
|---|---|
| `PRICES` | ⚠️ **가격 원장 (수정 주의)** — `[선박][박수][할인군][투어] = [주중,주말,성수기,독도주중,독도주말,독도성수기]` |
| `PEAK_RANGES` | 성수기 자동 추천 기간 (기본 `07-15 ~ 08-20`) |
| `TIERS` | 요금 등급 정의 및 기본 할인군 매핑 |
| `STAFF_TEL` | 담당자 이름 → 연락처 매핑 |
| `DEPOSIT_PER_PERSON` | 1인당 계약금 (기본 100,000원) |

### 계산 규칙 요약
- 도착일 = 출발일 + 박수 (1박2일 7/10 출발 → 7/11 도착)
- 혼합 노선(가는편≠오는편) = 두 선사 요금 평균 → 100원 단위 반올림
- 콤비투어 = 버스·택시 요금 평균
- 스노클링 = 렌트카투어 요금표(rents 열) 그대로 (+20,000원/인)
- 금액 = Σ(등급별 1인당 × 인원) + 숙소 추가요금
- 부가세: 카드/현금영수증 시 10% 별도 (문구에 자동 포함)

## 🛠️ 기술 스택

- **순수 HTML/CSS/JavaScript** — 프레임워크·빌드 도구·서버 없음
- [html2canvas](https://html2canvas.hertzen.com/) — 견적서 PNG 렌더링 (CDN)
- [Pretendard](https://github.com/orioncactus/pretendard) — 웹폰트 (CDN, 미로딩 시 시스템 폰트 폴백)
- `localStorage` — 자동 저장 · 견적 이력

## ✅ 품질 검증

- 가격 원장 576셀(3선사 × 1·2박 × 4할인군 × 4투어 × 6시즌칸)을 구버전과 **기계적 diff 전수 대조 — 완전 일치**
- 멀티에이전트 코드리뷰(계산 로직 / 런타임 / 모바일 UX 3개 관점 + 지적별 적대적 검증) 수행
  - 확정 지적 7건 전량 수정: HTML 주입 2건, 초기화 누락, 검증 데드코드, 터치 타깃 미달, 이력 오터치 데이터 소실 위험, 절취공 위치
- 구버전 대비 수정된 대표 버그: **도착일 하루 초과 계산**, deprecated 복사 API, 성수기 미감지, PNG 원격 이미지 의존
